Projeto criado para aplicar os meus conhecimentos em .NET utilizando o C#.

Atendendo os principais requisitos para vagas de emprego.

## Tecnologias utilizadas

- NET 7.0
- C#
- Markdig
- Razor pages
- SQLite
- Dapper
